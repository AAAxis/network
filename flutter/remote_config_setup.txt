FIREBASE REMOTE CONFIG SETUP GUIDE
===================================

STEP 1: Go to Firebase Console → Remote Config
STEP 2: Add the following parameters:

─────────────────────────────────────────────────────────────
PARAMETER 1: vpn_servers
─────────────────────────────────────────────────────────────
Type: String (JSON)
Default: [] (empty array)

Value: (Copy the entire content from remote_config_vpn_servers.json)

─────────────────────────────────────────────────────────────
PARAMETER 2: vpn_username
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)

─────────────────────────────────────────────────────────────
PARAMETER 3: vpn_password
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)

─────────────────────────────────────────────────────────────
PARAMETER 4: vpn_shared_secret
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)

─────────────────────────────────────────────────────────────
PARAMETER 5: revenuecat_api_key_android
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)
Example: goog_DMztEaCVhFMXcFodoKSmigHyUXu

─────────────────────────────────────────────────────────────
PARAMETER 6: revenuecat_api_key_ios
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)
Example: appl_mtyqcTcHlcOcNptWUraNaZCiJci

─────────────────────────────────────────────────────────────
PARAMETER 7: admob_app_id
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)
Example: ca-app-pub-9876848164575099~9384550593

─────────────────────────────────────────────────────────────
PARAMETER 8: admob_banner_ad_id
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)
Example: ca-app-pub-9876848164575099/9211599709

─────────────────────────────────────────────────────────────
PARAMETER 9: admob_interstitial_ad_id
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)
Example: ca-app-pub-9876848164575099/8393342054

─────────────────────────────────────────────────────────────
PARAMETER 10: admob_rewarded_ad_id
─────────────────────────────────────────────────────────────
Type: String
Default: (empty - MUST be set in Remote Config)
Example: ca-app-pub-9876848164575099/8393342054

─────────────────────────────────────────────────────────────
PARAMETER 11: interstitial_ads_enabled
─────────────────────────────────────────────────────────────
Type: Boolean
Default: true

─────────────────────────────────────────────────────────────
PARAMETER 12: reward_ads_enabled
─────────────────────────────────────────────────────────────
Type: Boolean
Default: true

─────────────────────────────────────────────────────────────
STEP 3: Click "Publish changes"
─────────────────────────────────────────────────────────────

After publishing, restart your app to fetch the new values.

⚠️ IMPORTANT: NO FALLBACKS EXPOSED
- All credentials and API keys MUST be set in Remote Config
- The app will fail to initialize if any required parameter is missing
- No hardcoded credentials or keys are exposed in the app code
